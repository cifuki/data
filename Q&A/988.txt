A:Currently, there is no widget provided in the Design Library. The only way to do it quickly and easily is by using third-party libraries. 

You can obviously do this using the Design Library too, but it will be a huge work and require a lot of time. I have mentioned some useful libraries, that can help you achieve this.


Rapid Floating Button (wangjiegulu)
Floating Action Button (Clans)
Floating Action Button (makovkastar)
Android Floating Action Button (futuresimple)


I am using the 4th one.
