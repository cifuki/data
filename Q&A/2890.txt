Q:Is it possible to make a phone number clickable in HTML so that when a user clicks on the number it is called?
    
===============================================================================================
150
A:Something like this should work:

<a href="tel:+1-800-555-5555">Call 1-800-555-5555</a>


More info:

http://en.wikipedia.org/wiki/URI_scheme#tel:
